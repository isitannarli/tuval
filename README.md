<p align="center">Tuval is a cross-platform app that lets you to draw on the desktop screen</p>

> Info: This app does not have a stable version yet. Please, let me know if you have a problem.


## Development

**Required:**

- NodeJS v18.12.0
- Rust v1.25.1

```shell
npm install
```

**Running**

```shell
npm run tauri dev
```

**Building**

```shell
yarn run tauri build
```
